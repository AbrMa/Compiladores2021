3. ¿Qué es el diseño orientado a objetos?
	Cuando para hacer que en la solución del problema, se describa al problema se usan a las características principales y elementales de las entidades relacionadas al problema para reducirlas a objetos y hacer que estos objetos interactuen con otros.

4. ¿Qué hace un AFD?
	Nos dice si una cadena pertenece o no a un leguaje, para cada símbolo sabe exactamente a que nuevo estado debe ir.

5. ¿Qué hace un AFN?
	Sirve también para ver si una cadena pertenece o no a un lenguaje, pero puede tener distintas transiciones para el mismo símbolo.
